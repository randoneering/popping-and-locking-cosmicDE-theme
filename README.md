# Popping and Locking Theme for COSMIC DE

A dark theme port for COSMIC DE, based on the popular "Popping and Locking" color scheme used in iTerm2, Ghostty, Zed, VSCode, and other development tools.

## Import into COSMIC DE

```
 # Clone Repo
 git clone https://github.com/randoneering/popping-and-locking-cosmicDE-theme.git
 
```

After cloning the repo, proceed to `Cosmic Settings` > `Desktop` > `Appearance` > `Import` and select the `.ron` file! 

## Color Palette

The theme uses a warm, earthy color palette with good contrast:

**Base Colors:**
- Background: #181921 (dark blue-gray)
- Foreground: #ebdbb2 (warm beige)
- Accent: #d79921 (golden yellow)

**Cursor and Selection:**
- Cursor: #c7c7c7 (light gray)
- Cursor Text: #8c8c8c (medium gray)
- Selection Background: #ebdbb2 (warm beige)
- Selection Foreground: #928374 (muted gray)

**ANSI Colors:**
- Black: #1d2021 / Bright: #928374
- Red: #cc241d / Bright: #f42c3e
- Green: #98971a / Bright: #b8bb26
- Yellow: #d79921 / Bright: #fabd2f
- Blue: #458588 / Bright: #99c6ca
- Magenta: #b16286 / Bright: #d3869b
- Cyan: #689d6a / Bright: #7ec16e
- White: #a89984 / Bright: #ebdbb2

## Features

- High contrast for readability
- Warm color temperature reduces eye strain
- Consistent with the original Popping and Locking theme across platforms [Zed](https://github.com/randoneering/popping-and-locking-zed-theme)
- Suitable for extended coding sessions
